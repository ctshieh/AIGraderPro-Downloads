如何驗證 SHA256？ 
# macOS / Linux
shasum -a 256 AIGraderPro_mac_arm64.dmg

# Windows (PowerShell)
Get-FileHash .\AIGraderPro_Windows_x64.zip -Algorithm SHA256

# 比對輸出與 .sha256 檔內容一致即可
